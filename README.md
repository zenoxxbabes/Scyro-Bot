# 🤖 Scyro

Scyro is a modern and customizable **Discord bot written in Python**.  
It provides useful utilities, moderation tools, and interactive features for Discord servers.

This project is now **open source**, so developers can explore the code, modify it, and build their own versions.

# ✨ Features

- ⚙️ Utility commands
- 🛡️ Moderation tools
- 🎮 Fun commands
- ⚡ Fast and lightweight
- 🔧 Easy configuration
- 🌐 Optional website integration

# 🛠 Installation

### 1 Clone the repository

```
git clone https://github.com/zenoxxbabes/Scyro-Bot.git
```

### 2 Go into the project directory

```
cd Scyro-Bot
```

### 3 Install dependencies

```
pip install -r requirements.txt
```

### 4 Configure the bot
```
rename `example.env` to `.env` and put the credentials.
```

Example:

```
TOKEN=your_discord_bot_token
```

### 5 Run the bot

```
python main.py
```
